### Framework Candidates

- [ ] **Ingestion PayloadValidator / ValidationResult** — Minoo has structured validation; Claudriel has ad-hoc validation in handlers. If both apps validate ingest payloads, the validator pattern should be framework code.
- [ ] **Per-domain provider splitting** — Waaseyaa skill says "one provider per domain." Minoo follows this. If the framework enforced or scaffolded this, it would prevent Claudriel's monolithic drift.
- [ ] **Deployer recipe** — Both apps use nearly identical `deploy.php` with artifact upload + migration + cache clear + FPM reload. Extract a shared Deployer recipe to `waaseyaa/deployer`.
