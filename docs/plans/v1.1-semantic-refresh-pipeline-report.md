  - preserved published-only visibility behavior for `node`,
  - preserved stable aggregate report format for `warm()`.
- Added resumable batch refresh API:

  - normalized entity type ordering by input sequence,

- Missing entity types remain non-fatal (`completed_with_skips`) with explicit per-type reporting.
- No-provider behavior is explicit (`skipped_no_provider`) and returns non-zero exit from CLI.
