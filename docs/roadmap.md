# Waaseyaa Roadmap

Issue tracking and milestone planning is managed on GitHub:

- **Milestones:** https://github.com/jonesrussell/waaseyaa/milestones
- **Open issues:** https://github.com/jonesrussell/waaseyaa/issues

## Milestones

| Milestone | Description |
|---|---|
| [v0.1 — Framework Identity](https://github.com/jonesrussell/waaseyaa/milestone/1) | Kernel, skeleton, config-driven entity types, CLI, publishing automation |
| [v0.2 — Auth & Storage](https://github.com/jonesrussell/waaseyaa/milestone/2) | JWT/API key auth, disk-backed file storage, file upload, dev server stability |
| [v0.3 — Native SSR Engine](https://github.com/jonesrussell/waaseyaa/milestone/4) | Twig rendering pipeline, field formatters, view modes, public routing, theme system |
| [v0.4 — AI & MCP](https://github.com/jonesrussell/waaseyaa/milestone/5) | Vector embeddings, semantic search, MCP endpoint — AI layer online before the content model |
| [v0.5 — Diidjaaheer](https://github.com/jonesrussell/waaseyaa/milestone/3) | Cultural groups, teachings, events, articles, Ojibwe language model |
| [v1.0 — Production](https://github.com/jonesrussell/waaseyaa/milestone/6) | Performance hardening, security audit, FrankenPHP config, Telescope, deployment docs |
